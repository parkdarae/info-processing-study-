# 브라우저 캐시 완전 삭제 방법

이미지 표시 문제를 해결하기 위해 브라우저 캐시를 완전히 삭제해주세요.

## 방법 1: 브라우저에서 직접 삭제 (권장)

1. **브라우저에서 `Ctrl + Shift + Delete` 키를 누르세요**
2. **시간 범위**: "전체 기간" 선택
3. **삭제할 항목**:
   - ✅ 쿠키 및 기타 사이트 데이터
   - ✅ 캐시된 이미지 및 파일
4. **"데이터 삭제" 버튼 클릭**

## 방법 2: force_clear.html 사용 (자동)

1. `start_server.bat` 실행 (이미 실행 중이면 생략)
2. 브라우저에서 `http://localhost:8000/force_clear.html` 접속
3. 자동으로 캐시가 삭제되고 메인 페이지로 이동합니다

## 캐시 삭제 후 확인 사항

1. `http://localhost:8000` 접속
2. **2025년 1회** 선택
3. **순차 풀기** 클릭
4. **Q002 문제** 확인 - 이미지가 정상적으로 표시되어야 함
5. **F12** 눌러서 Console 탭 확인 - 에러가 없어야 함

## 문제가 계속되는 경우

- 브라우저 완전 종료 후 다시 시작
- 시크릿/프라이빗 모드에서 테스트
- 다른 브라우저에서 테스트 (Chrome, Edge, Firefox 등)
