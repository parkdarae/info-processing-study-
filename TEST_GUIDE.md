## ✅ 테스트 체크리스트

### 1. 메뉴 확인
- [ ] 햄버거 메뉴 열기
- [ ] "PMP 문제집" 항목 확인
- [ ] 클릭 시 PMP 대시보드 표시

### 2. 대시보드 확인
- [ ] 총 문제 수: 798개 표시
- [ ] 지식영역 10개 카테고리 표시
- [ ] 프로세스 그룹 5개 카테고리 표시
- [ ] "전체 문제 학습" 버튼 동작

### 3. 문제 화면 확인
- [ ] 문제 번호 및 라벨 표시
- [ ] **"강조 ON" 버튼 존재 확인** ⭐
- [ ] "체크" 버튼 동작
- [ ] 4개 선택지 표시

### 4. 키워드 강조 기능 (핵심!)

- [ ] 주요 키워드가 **보라색 굵은글씨**로 표시
  - 예: 프로젝트 관리자, 애자일, 스프린트, 통합, 범위 등
- [ ] "강조 OFF" 버튼으로 원래 상태 복귀
- [ ] 문제 본문과 해설 모두 적용 확인

### 5. 문제 풀이 확인
- [ ] 선택지 클릭 시 하이라이트
- [ ] "정답 확인" 버튼 동작
- [ ] "답 보기" 버튼으로 정답 표시
- [ ] "해설 보기" 버튼으로 해설 표시
  - [ ] 해설에도 키워드 강조 적용 확인

### 6. 네비게이션 확인

- [ ] "다음 문제" 버튼 동작
- [ ] "대시보드" 버튼으로 복귀

### 7. 학습 모드 확인
- [ ] 순차 학습
- [ ] 랜덤 학습

### 8. 데이터 품질 확인

## 🐛 예상 이슈 및 해결

### 이슈 1: 한글 깨짐
**원인**: UTF-8 인코딩 문제
**해결**: Python 서버 사용 또는 브라우저 인코딩 설정

### 이슈 2: 키워드 강조 안됨
**확인사항**:
1. "강조 ON" 버튼을 클릭했는가?
2. 브라우저 캐시 삭제 (Ctrl+F5)
3. js/pmp-module.js 파일 최신 버전 확인

### 이슈 3: 문제 로드 안됨
**확인사항**:
1. data/items_pmp.jsonl 파일 존재 확인
2. 파일 크기 확인 (약 1.5MB)
3. 브라우저 콘솔 에러 확인 (F12)

## 🎯 주요 테스트 시나리오

### 시나리오 1: 기본 학습
1. PMP 대시보드 접속
2. "전체 문제 학습" 클릭
3. 문제 1번 표시 확인
4. "강조 ON" 클릭
5. 키워드 강조 확인
6. 선택지 선택 → 정답 확인
7. 해설 보기 → 키워드 강조 확인
8. 다음 문제로 이동

### 시나리오 2: 카테고리별 학습
1. 대시보드에서 "통합 관리" 선택
2. 해당 카테고리 문제만 표시 확인
3. 강조 기능 동작 확인

### 시나리오 3: 체크 기능
1. 문제에서 "체크" 버튼 클릭
2. 대시보드로 복귀
3. "체크한 문제 학습" 클릭
4. 체크한 문제만 표시 확인
