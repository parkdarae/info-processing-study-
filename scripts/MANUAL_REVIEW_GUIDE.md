## ✍️ 해설 작성 가이드

### 좋은 해설의 특징

1. **초기 상태 명시**
   ```
   초기값: a=0, b=5, sum=0
   ```

2. **단계별 실행 추적** (표 형식 권장)
   ```
   | 반복 | 조건 | 실행 | 결과 |
   |------|------|------|------|
   | i=0  | true | a+=i | a=0  |
   | i=1  | true | a+=i | a=1  |
   ```

3. **중간 계산 과정 표시**
   ```
   i=2:
   - b[i] = a[i] * 2 = 4 * 2 = 8
   - sum = sum + b[i] = 3 + 8 = 11
   ```

4. **최종 결과 강조**
   ```
   정답: 22
   ```

5. **핵심 개념 정리**
   ```
   핵심:
   - 포인터 연산: p+i는 &a[i]와 동일
   - 후위 증감: i++는 사용 후 증가
   ```
